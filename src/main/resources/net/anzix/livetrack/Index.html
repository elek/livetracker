Livetrack

This application is a simple demo live tracker for locus (and for any other application.)

  • Registration are not required you can use any string as a key of one tracking.
  • The application stores only the latest position for a specific key, and it could be shown on the map.
  • The store is not persistent. Every data will be deleted with the next restart.
  • All pages are public and anonym.

API

UPDATE

curl --data "lat=47&lon=19" ${url}api/point/{MAP_KEY}/{CLIENT_KEY}

GET

curl ${url}api/point/{MAP_KEY}

MAP

${url}map/{MAPKEY}

Note: Use any string instead of {MAP_KEY}. As every page is public: use complex keys if you need pseudo-private page. CLIENT_KEY also could be any string. It identifies the client, so more than one client could post positions to the same map.
